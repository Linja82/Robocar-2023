# Arduino Pins
- D0: 
- D1:
- D2: Mode Select Switch
- D3: Motor Standby
- D4: LED Data
- D5: Right Motor PWM
- D6: Left Motor PWM
- D7: Right Motor Direction
- D8: Left Motor Direction
- D9: IR Receiver
- D10: Servo 1
- D11: Servo 2
- D12: Ultrasonic Echo
- D13: Ultrasonic Trigger
---------------------
- A0: Right Line Tracker
- A1: Middle Line Tracker
- A2: Left Line Tracker
- A3: Battery Voltage
- A4: MPU6050 SDA
- A5: MPU6050 SCL
